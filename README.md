# CHANGe UP (팀명: 바지사장)

## 서비스 소개
* 서비스명:  창업 정보 제공 웹사이트
* 서비스설명: 다양한 창업 정보를 제공하고 AI를 이용하여 창업 아이템을 추천해주는 서비스입니다.
<br>

## 📅 프로젝트 기간
2024.06.12 ~ 2024.06.25 (2주)
<br>

## ⭐ 주요 기능
* AI 창업 추천
* 창업 정보 제공
<br>
